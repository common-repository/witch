== Description ==
WordPress moTion Controlling enables your visitors to use motion gestures to interact with your page.

View a Demo [here](http://youtu.be/4Y-GLTlUgvQ)!

At the moment no options to configure.


= Features =
* use right and left swipe to navigate between posts

= Supported Controllers =
* [Leap Motion 3D Controller](https://www.leapmotion.com/) 

== Installation ==
1. Upload the witch folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
